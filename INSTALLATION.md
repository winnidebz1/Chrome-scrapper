## üêõ Troubleshooting

### Extension Won't Load
- Make sure you selected the correct folder
- Check that all files are present (manifest.json, popup.html, etc.)
- Try reloading: Click the refresh icon on the extension card

### Scan Button Doesn't Work
- Make sure you're on Google Maps (https://www.google.com/maps)
- Make sure you've searched for businesses (not just viewing the map)
- Check the browser console for errors (F12 ‚Üí Console)

### No Results Appear
- Try scrolling down on Google Maps to load more businesses
- Some searches may have businesses with websites (try different searches)
- Check filters - make sure "No Website Only" isn't filtering everything out

### Console Errors
1. Right-click the extension icon
2. Select "Inspect popup"
3. Check the Console tab for errors
4. Report errors in GitHub Issues

## üéØ Best Practices for Finding Leads

### High-Quality Searches

**Good Examples:**
- "Hair salons in [City]"
- "Restaurants in [Neighborhood]"
- "Auto repair shops near me"
- "Plumbers in [City]"

**Why These Work:**
- Local businesses

- Often lack websites
- High volume of results

### Industries with High No-Website Rates

1. **Personal Services** - Salons, barbershops, spas
2. **Food & Beverage** - Small restaurants, cafes, food trucks
3. **Automotive** - Repair shops, detailing, mechanics
4. **Home Services** - Plumbers, electricians, handymen
5. **Retail** - Small shops, boutiques, local stores

### Geographic Targeting

**Best Locations:**
- Developing countries (higher no-website rate)
- Small towns and suburbs
- Ethnic neighborhoods
- Industrial areas
